# AI‑Справочник городского хозяйства

Интерактивный веб‑справочник по вопросам:
- ЖКХ
- экологии
- благоустройства

Проект создан с использованием HTML, CSS и JavaScript.

- интерактивный интерфейс;
- ответы на вопросы пользователя;
- работа в браузере;
- публикация через GitHub Pages.
